# yesimthisgoofy

live at → **https://yesimthisgoofy.netlify.app**

a little something for someone. two buttons:

- **surprise me 💘** — a random date idea, weighted by time of day
- **find me taco bell 🌮** — your browser location + OpenStreetMap → nearest Taco Bell within 25km, with an "open in maps" button

single self-contained `index.html`. no build step, no npm, no backend.

## updating

edit `index.html`, then:

```bash
git commit -am "update"
git push
```

netlify redeploys automatically in ~10 seconds.
